# Assignment: your minimal site

1. Choose **Roblox-inspired** *or* **Labubu-inspired** starter (or mix ideas — e.g. dark mode + chunky borders — but keep it **one page**).
2. Personalize:
   - Page `<title>` and main `<h1>`
   - At least **two** sections (e.g. hero + about, or gallery + links)
   - At least **one** real link (`<a href="...">`) opening in a new tab where appropriate (`target="_blank"` + `rel="noopener noreferrer"`)
3. Use only **HTML + Tailwind** (no JavaScript required).
4. Optional: add a **favicon** later; not required for this lesson.
