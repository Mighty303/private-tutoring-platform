# Exercise 5: K Largest Elements 📏
`🎯 Functions` `📋 Lists` `🔁 Loops`

**Your Task:**
Write a function `k_largest(nums, k)` that:
- Returns the k largest elements from the list
- Returns them in descending order
- Use a heap!

**Examples:**
```python
print(k_largest([5, 3, 8, 1, 4, 2, 7], 3))
# Output: [8, 7, 5]

print(k_largest([10, 20, 30, 40], 2))
# Output: [40, 30]
```

**Hint:** Use the max-heap trick (negate values)!
