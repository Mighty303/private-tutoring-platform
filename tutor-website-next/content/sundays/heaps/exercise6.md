# Exercise 6: Sort Using a Heap (Heap Sort) 📊
`🎯 Functions` `📋 Lists` `🔁 Loops`

**Your Task:**
Write a function `heap_sort(nums)` that:
- Returns a new sorted list (ascending)
- Uses a heap to sort!
- Do NOT use Python's built-in `sort()`

**Examples:**
```python
print(heap_sort([5, 3, 8, 1, 4, 2, 7]))
# Output: [1, 2, 3, 4, 5, 7, 8]

print(heap_sort([10, 1, 10, 1]))
# Output: [1, 1, 10, 10]
```

**Hint:** Heapify, then pop one by one!
