# Intro to Web Development

You will build a **very small website** in one file on your laptop: `index.html`. You will use **HTML** for structure and **Tailwind CSS** for styling. Tailwind gives you ready-made class names (like `text-xl` and `bg-red-600`) so you can style pages without writing a separate CSS file at first — that makes it easier to teach and to experiment.

At the end, pick **one** theme inspired by **Roblox** (bold, high-energy, red vs cyan) or **Labubu / The Monsters** (soft panels, thick outlines, playful colors) and make it your own.

This track is split into **short pages** so you can read one section at a time without feeling overwhelmed.

---

## What you need

- A laptop with **Chrome**, **Edge**, **Firefox**, or **Safari**
- A text editor (**VS Code** recommended — free)
- This lesson open while you type the code snippets

---

## How a web page works (short version)

| Piece | Job |
|--------|-----|
| **HTML** | Describes *what* is on the page: headings, paragraphs, images, links, sections |
| **CSS** | Describes *how* it looks: colors, spacing, fonts, layout |
| **Browser** | Reads your HTML, applies CSS, and draws the page |

**Tailwind** is a **CSS framework**: you add special class names to your HTML, and Tailwind turns those into real CSS behind the scenes. For class, you will load Tailwind from a **CDN** (a fast copy on the internet) with one `<script>` tag — no install required.

---

## Create your project file

1. Make a new folder on your computer, e.g. `my-first-site`.
2. Inside it, create a file named **`index.html`** (exactly that name).
3. Open `index.html` in your editor and in your browser: double-click `index.html` in Finder / File Explorer to open it in the default browser, or open it from your editor’s file tree.

When you save changes and refresh the browser (**Ctrl+R** or **Cmd+R**), you will see updates.
