# Exercise 3: Find the Minimum in a List

`📋 Lists` `🔂 Loops`

## Your Task

Write a program that finds the **minimum** (smallest) number in a list. **Do not use the built-in `min()` function** — implement it yourself with a loop.

**Example:** `[7, 2, 9, 1, 5, 3]` → minimum is `1`

Use a variable to track the smallest value seen so far as you loop through the list.

---

## Example Output

```
List: [7, 2, 9, 1, 5, 3]
Minimum: 1
```
