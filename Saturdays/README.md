# Python Tutoring for Grade 7 🐍

Teaching Python fundamentals to grade 7 girls through gamified, Roblox-themed exercises.

---

## 📂 Repository Contents

### Current Materials

- **`Cheatsheet.md`** - Quick reference guide for all Python basics (variables, loops, functions, etc.)
- **`Review.md`** - 8 hands-on exercises with break-time rewards (2-hour lesson plan)
- **`Functions.md`** - Advanced function practice with problem-solving framework
- **`function.py`** - Example Python code/solutions

---

## 🎯 What This Covers (Current Milestone)

**Milestone 1: Python Fundamentals** ✅

Students learn:
- Variables and data types
- Type casting (converting between types)
- Operators (math, comparison, logical)
- User input
- Strings (f-strings, slicing, methods)
- Lists (append, remove, indexing)
- Conditionals (if/elif/else)
- While loops
- For loops
- Functions (basics)

**Time:** 4-6 lessons (2 hours each)

## 📅 Future Milestones

### Milestone 2: Intermediate Python (Next)
**Estimated: 4-5 lessons**

- **Dictionaries** - Store player data with key-value pairs
- **List comprehensions** - Create lists in one line
- **String formatting** - Advanced text manipulation
- **Error handling** - Try/except blocks
- **File I/O** - Save and load game data
- **Modules** - Import and use libraries

**Project Ideas:**
- Save/load player profiles to files
- Create a contact list app
- Build a word frequency counter

---

### Milestone 3: Game Development
**Estimated: 6-8 lessons**

- **Classes & Objects** - Create player and enemy characters
- **Inheritance** - Different enemy types from base class
- **Game loops** - Main game logic
- **Random module** - Dice rolls, treasure drops
- **Time module** - Delays and timing
- **Simple graphics** - Turtle or Pygame basics

**Project Ideas:**
- Text-based RPG with save system
- Simple platformer game
- Quiz game with leaderboard

---

### Milestone 4: Real-World Applications
**Estimated: 4-6 lessons**

- **APIs** - Get data from websites
- **Web scraping** - Extract information from web pages
- **Data analysis** - Work with CSV files
- **Discord bots** - Create simple bot commands
- **Automation** - Simple scripts to automate tasks

**Project Ideas:**
- Weather checker app
- Discord bot with custom commands
- Roblox stats tracker
- Minecraft server status checker

---

### Milestone 5: Advanced Projects
**Estimated: 8-10 lessons**

- **GUI applications** - Tkinter basics
- **Databases** - SQLite for data storage
- **Algorithms** - Sorting and searching
- **Data structures** - Stacks, queues, trees
- **Final project** - Student choice

**Project Ideas:**
- Inventory management system

- Personal finance tracker
- Mini social media app

## 📊 Progress Tracking

**Beginner** → Completes Review exercises 1-5  
**Intermediate** → Completes all 8 Review exercises  
**Advanced** → Completes Bonus challenge + Functions exercises  
**Expert** → Creates own Roblox-themed exercise

---

## 🎯 Key Learning Goals

By the end of Milestone 1, students should:
- ✅ Write programs without syntax errors
- ✅ Use proper indentation and colons
- ✅ Convert user input correctly
- ✅ Work with lists and loops confidently
- ✅ Create simple functions
- ✅ Debug their own code
- ✅ Help peers with concepts
